📚 큰수의 법칙이란?

큰수의 법칙(Law of Large Numbers)은 확률론의 기본 원리 중 하나로, 같은 실험을 반복할수록 실제 결과가 이론적 확률에 가까워진다는 법칙입니다.

🎯 주사위 예시

이론적 확률: 주사위를 무한히 던지면 각 면이 나올 확률은 1/6 ≈ 16.67%
이론적 평균: (1+2+3+4+5+6) ÷ 6 = 3.5
큰수의 법칙: 던지는 횟수가 늘어날수록 실제 평균이 3.5에 가까워집니다.

🔬 극한을 통한 수학적 정의

큰수의 약한 법칙 (Weak Law of Large Numbers):
주사위를 n번 던졌을 때의 평균을 X̄n이라 하면,
limn→∞ P(|X̄n - 3.5| > ε) = 0
즉, 아무리 작은 양수 ε에 대해서도, n이 충분히 크면 실제 평균과 이론적 평균의 차이가 ε보다 클 확률이 0에 수렴합니다.

📊 극한의 직관적 이해

ε-δ 논법으로 설명하면:
• 임의의 작은 오차 ε > 0이 주어졌을 때
• 충분히 큰 자연수 N이 존재하여
• n ≥ N일 때 |X̄n - 3.5| < ε
• 즉, n이 N보다 크면 실제 평균이 3.5 ± ε 범위 안에 들어갑니다.

🎲 시뮬레이션에서 확인할 수 있는 것

1. 수렴성: 던진 횟수가 늘어날수록 실제 평균이 3.5에 수렴
2. 안정성: 충분히 많은 횟수 후에는 평균이 크게 변하지 않음
3. 극한의 의미: "무한히 던진다면" 실제 평균 = 3.5
4. 오차의 감소: n이 커질수록 |X̄n - 3.5| → 0

🔢 부등호 변환 과정 (교과서 기반)

원래 부등식: |X/n - 1/6| < 0.1

단계별 변환:
1단계: |X/n - 1/6| < 0.1
2단계: -0.1 < X/n - 1/6 < 0.1
3단계: -0.1 + 1/6 < X/n < 0.1 + 1/6
4단계: -3/30 + 5/30 < X/n < 3/30 + 5/30
5단계: 2/30 < X/n < 8/30
6단계: 1/15 < X/n < 4/15
7단계: n/15 < X < 4n/15

부등호 방향: 모든 단계에서 < (미만) 또는 > (초과)로 유지됩니다!

📊 교과서 계산 예시 (교과서 표 기반)

n=10일 때:
10/15 < X < 4×10/15 → 2/3 < X < 8/3 → X = 1, 2
P(|X/10 - 1/6| < 0.1) = P(X=1) + P(X=2) = 0.3230 + 0.2907 = 0.6137

n=30일 때:
30/15 < X < 4×30/15 → 2 < X < 8 → X = 3, 4, 5, 6, 7
P(|X/30 - 1/6| < 0.1) = P(X=3) + P(X=4) + ... + P(X=7) = 0.7835

n=50일 때:
50/15 < X < 4×50/15 → 3.33 < X < 13.33 → X = 4, 5, ..., 13
P(|X/50 - 1/6| < 0.1) = P(X=4) + P(X=5) + ... + P(X=13) = 0.9455

교과서 결론: n이 커질 때, 확률 P(|X/n - 1/6| < 0.1)은 1에 가까워집니다!

📚 교과서: 큰수의 법칙의 정확한 정의

큰수의 법칙:
어떤 시행에서 사건 A가 일어날 수학적 확률이 p일 때, n번의 독립시행에서 사건 A가 일어나는 횟수를 X라고 하면
임의의 작은 양수 h에 대하여 n이 한없이 커질 때, 확률 P(|X/n - p| < h)는 1에 한없이 가까워진다.

주사위 예시에서:
• 사건 A: 3의 눈이 나오는 사건
• 수학적 확률 p: 1/6 ≈ 0.1667
• 상대도수 X/n: 실제로 3이 나온 비율
• h: 허용 오차 (0.1, 0.01, 0.001 등)

의미: 시행 횟수 n이 충분히 클 때, 상대도수 X/n는 수학적 확률 1/6에 가까워집니다.

🎯 교과서: 큰수의 법칙의 활용

1. 통계적 확률의 근사:
시행 횟수 n이 충분히 클 때, 상대도수 X/n는 수학적 확률에 가까워지므로
사건 A의 상대도수 X/n를 사건 A가 일어날 확률 P(A)로 쓸 수 있습니다.

2. 자연현상과 사회현상:
수학적 확률을 구하기 곤란한 경우에는 시행 횟수를 충분히 크게 한 후
통계적 확률을 이용할 수 있습니다.

3. 시뮬레이션의 의미:
